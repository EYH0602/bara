# ara-cli

The command-line runtime for the [ARA viewer](https://github.com/EYH0602/bara).
Installs a binary named `ara`.

```bash
cargo install ara-cli
ara --help
```

Planned commands: `ara validate <dir>` and `ara serve <dir>`. Skeleton
reservation release — these land in a later version.
